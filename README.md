# Анализатор текста

**Графическое приложение для анализа текстовых файлов с поддержкой Drag & Drop.**

---

## Описание

Этот проект предоставляет удобный графический интерфейс для анализа текстовых файлов. Приложение позволяет:
- Загружать текстовые файлы через диалог выбора или перетаскиванием (Drag & Drop).
- Анализировать текст по различным параметрам: количество символов, слов, частотность букв и слов.
- Фильтровать результаты анализа (только буквы, только слова или всё вместе).
- Сохранять результаты анализа в формате JSON.

---

## Возможности

- **Загрузка файлов** — через диалог или перетаскиванием.
- **Анализ текста** — подсчёт символов, слов, частотность букв и слов.
- **Фильтрация результатов** — выбор категорий анализа.
- **Сохранение результатов** — в формате JSON.
- **Удобный интерфейс** — с вкладками для результатов и содержимого файла.

---

## Установка

### Требования
- Python 3.6+
- Библиотеки: `tkinter`, `tkinterdnd2`

### Установка зависимостей
```bash
pip install tkinterdnd2
```

---

## Использование

### Запуск приложения:
```bash
python main.py
```

### Загрузка файла:
Нажмите "Выбрать файл" или перетащите файл в область Drag & Drop.

### Анализ текста:
Нажмите "Выполнить анализ".
Выберите фильтры анализа (все категории, только буквы или только слова).

### Просмотр результатов:
Результаты анализа отобразятся в дереве на вкладке "Результаты анализа".
Содержимое файла доступно на вкладке "Содержимое файла".

### Сохранение результатов:
Нажмите "Сохранить в JSON", чтобы сохранить результаты анализа.
